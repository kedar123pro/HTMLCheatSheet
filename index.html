A

The a tag defines a hyperlink, which is used to link from one page to another.
The most important attribute of the < a > element is the href attribute, which indicates the link's destination.

Attributes:

  1. charset - Specifies the character-set of a linked document
  2. coords - Specifies the coordinates of a link
  3. download - Specifies that the target will be downloaded when a user clicks on the hyperlink
  4. href - Specifies the URL of the page the link goes to
  5. hreflang - Specifies the language of the linked document
  6. media - Specifies what media/device the linked document is optimized for
  7. name - Specifies the name of an anchor
  8. rel - Specifies the relationship between the current document and the linked document
  9. rev - Specifies the relationship between the linked document and the current document
  10. shape - Specifies the shape of a link
  11. target - Specifies where to open the linked document
  12. type - Specifies the MIME type of the linked document

input

  9. formaction - Specifies the URL of the file that will process the input control when the form is submitted, for type="submit" and type="image"
  10. formenctype - Specifies how the form-data should be encoded when submitting it to the server, for type="submit" and type="image"
  11. formmethod - Defines the HTTP method for sending data to the action URL for type="submit" and type="image"
  12. formonvalidate - Defines that form elements should not be validated when submitted
  13. formtarget - Specifies where to display the response that is received after submitting the form for type="submit" and type="image"
  14. height - Specifies the height of an input element only for type="image"
  15. list - Refers to a datalist element that contains pre-defined options for an input element
  16. max - Specifies the maximum value for an input element
  17. maxlength - Specifies the maximum number of characters allowed in an input element
  18. min - Specifies a minimum value for an input element
  19. multiple - Specifies that a user can enter more than one value in an input element
  20. name - Specifies the name of an input element
  21. pattern - Specifies a regular expression that an input element's value is checked against
  22. placeholder - Specifies a short hint that describes the expected value of an input element
  23. readonly - Specifies that an input field is read-only
  24. required - Specifies that an input field must be filled out before submitting the form
  25. size - Specifies the width, in characters, of an input element
  26. src - Specifies the URL of the image to use as a submit button, only for type="image"
  27. step - Specifies the legal number intervals for an input field
  28. type - Specifies the type input element to display
  29. value - Specifies the value of an input element
  30. width - Specifies the width of an input element, only for type="image"
